如何使用hugo 搭建自己blog

本地运行

hugo server --theme=hyde --buildDrafts

访问： http://localhost:1313/blog/ (因为地址是https://clsss.github.io/blog/)

部署

  1. 在github创建仓库并复制地址

  2. 在根目录下执行命令，所有静态页面都会生成到 public 目录

hugo --theme=hyde --baseUrl="https://clsss.github.io/blog"

如果将theme，baseUrl写入了config.toml 则直接运行

hugo

提交到github

cd public
git add .
git commit -m "描述"
git push

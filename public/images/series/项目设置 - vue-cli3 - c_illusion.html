项目名
注意：修改项目名将导致项目的访问 URL（包含 Git 仓库的 URL）改变，在此之前的 URL 地址将失效。Git 仓库地址修改方法：git remote set-url origin [NEW_URL]项目描述
项目转让
项目转让后，你将不再是“项目所有者”，你将失去某些管理权限（如添加成员、删除项目等）。同时，项目转让后，项目的 Git 仓库地址会发生变化，例如：用户 Coding 将项目转给用户 Demo 后， Git 仓库访问路径将由 https://git.coding.net/coding/test.git 变成 https://git.coding.net/demo/test.git
您的项目还没添加成员
项目归档
项目归档后，该项目将会移到“项目所有者”的归档项目中，并且不能对该项目做任何操作，直到解除归档。同时该项目将从所有项目成员的项目列表中移除。“项目所有者”可以在项目->归档项目中进行解除归档操作，解除归档后，项目成员仍保留。
删除项目
删除项目将会连同其相关的所有数据（包括任务、合并请求、动态等）一起删除，此操作无法恢复！
